玩法：                                                                                                                
  - ↑ ↓ ← → 或 W A S D 控制方向                                                                                         
  - 吃红色食物增长，撞墙或撞自己则游戏结束                                                                              
  - 每吃 5 个食物速度提升一次                                                                                         
  - Space 或 P 暂停/继续                                                                                                
  - 填满整个格子算通关                                                                                                  
                                                                                                                        
  视觉特点：                                                                                                            
  - 蛇身渐变颜色（绿→青）                                                                                               
  - 蛇头带眼睛，会跟随朝向                                                                                              
  - 食物带呼吸光效                                                                                                      
  - 棋盘格背景                                                                                                          
                                                                                                                        
  直接打开 index.html 即可游玩。
